Определение математической статистики

Раздел математики, разрабатывающий методы сбора, систематизации, обработки и использования статистических данных.

Основные задачи:

  • Оценка неизвестных параметров
  • Проверка статистических гипотез
  • Установление взаимосвязей
  • Прогнозирование на основе данных

Выборка

Конечный набор объектов, случайно отобранных из генеральной совокупности.

Пример 1:

Рост студентов (см):

[175, 168, 182, 170, 165]

Пример 2:

Оценки студентов:

[5, 4, 3, 5, 4, 4]

Типы выборок:

  • Простая случайная
  • Стратифицированная
  • Систематическая
  • Кластерная

Выборочная средняя

Сумма всех элементов выборки, делённая на их количество.

$$\bar{x} = \frac{1}{n}\sum_{i=1}^{n} x_i$$

Пример:

[5, 8, 3, 9, 5]

$$\frac{5 + 8 + 3 + 9 + 5}{5} = \frac{30}{5} = 6$$

Среднее = 6

Медиана

Значение, которое делит упорядоченную выборку на две равные части.

Пример:

[5, 8, 3, 9, 5]

Сортируем: [3, 5, 5, 8, 9]

Средний элемент: 5

Мода

Значение, которое встречается в выборке наиболее часто.

Пример:

[5, 8, 3, 9, 5, 5]

Частоты: 5→3, 8→1, 3→1, 9→1

Мода = 5

Минимум и максимум

Минимум — наименьшее значение.

Максимум — наибольшее значение.

Пример:

[5, 8, 3, 9, 5]

Минимум = 3

Максимум = 9

Размах

Разность между максимальным и минимальным значениями.

$$R = x_{\max} - x_{\min}$$

Пример:

[5, 8, 3, 9, 5]

$$\text{max} = 9, \quad \text{min} = 3$$

Размах = $$9 - 3 = 6$$

Дисперсия

Среднее арифметическое квадратов отклонений от среднего.

$$s^2 = \frac{1}{n}\sum_{i=1}^{n} (x_i - \bar{x})^2$$

Пример:

[5, 8, 3, 9, 5], среднее = 6

Квадраты отклонений: 1, 4, 9, 9, 1

Сумма = 24, Дисперсия = $$\frac{24}{5} = 4.8$$

Нормальное распределение

Распределение Гаусса, график имеет форму колокола.

$$f(x) = \frac{1}{\sigma\sqrt{2\pi}} e^{-\frac{(x-\mu)^2}{2\sigma^2}}$$

Примеры:

  • Рост людей
  • Ошибки измерений
  • IQ показатели

JavaScript: Выборочная средняя

function calculateMean(data) {
    if (data.length === 0) return 0;
    const sum = data.reduce((a, b) => a + b, 0);
    return sum / data.length;
}

// Пример использования
const numbers = [5, 8, 3, 9, 5];
const mean = calculateMean(numbers);
console.log(mean); // 6

JavaScript: Медиана

function calculateMedian(data) {
    if (data.length === 0) return 0;
    const sorted = [...data].sort((a, b) => a - b);
    const middle = Math.floor(sorted.length / 2);
    
    if (sorted.length % 2 === 0) {
        return (sorted[middle - 1] + sorted[middle]) / 2;
    } else {
        return sorted[middle];
    }
}

JavaScript: Дисперсия

function calculateVariance(data) {
    if (data.length === 0) return 0;
    
    const mean = calculateMean(data);
    const sumSquaredDiffs = data.reduce((sum, value) => {
        return sum + Math.pow(value - mean, 2);
    }, 0);
    
    return sumSquaredDiffs / data.length;
}

Математическое ожидание

Среднее значение случайной величины.

$$E[X] = \sum_{i=1}^{n} x_i \times p_i$$

Пример: Игральная кость

Возможные значения: 1, 2, 3, 4, 5, 6

Вероятности: $$p_i = \frac{1}{6}$$

$$E[X] = \frac{1+2+3+4+5+6}{6} = 3.5$$

Дисперсия случайной величины

Мера разброса значений относительно мат. ожидания.

$$D[X] = E[X^2] - (E[X])^2$$

Пример: Игральная кость

$$E[X] = 3.5$$

$$E[X^2] = \frac{1+4+9+16+25+36}{6} \approx 15.167$$

$$D[X] = 15.167 - 3.5^2 = 2.917$$

Стандартное отклонение

Корень квадратный из дисперсии.

$$\sigma = \sqrt{D[X]}$$

Пример:

Для выборки [5, 8, 3, 9, 5]:

Дисперсия = 4.8

Стандартное отклонение = $$\sqrt{4.8} \approx 2.19$$
